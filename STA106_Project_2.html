Diagnostics

**Conduct Diagnostic Tests using A*B interaction effects model**

_ Shapiro-Wilk Test Brown-Forsythe Test
p-value 0.0273 0.4267

We use the A*B interaction effects model for diagnostics and first plot a QQplot of the residuals (ei) in order to superficially assess the normality of the errors assumption. The QQplot suggests that the data appear to be distributed approximately normal as most points lie near or on the projected line that indicates the plot of a completely normal dataset. There is a collection of points at the left tail that deviate slightly above the projected line, indicating that multiple low value sample BECK scores are slightly larger than what they would be under a completely normal distribution. The plot suggests that the most negative residual (left-most) point may be an outlier but we need to more rigorously use semi-studentized and studentized residual cutoffs to make a stronger case.

From the boxplot by treatment groups, it appears that the variances in BECK score of all treatment groups appear to be approximately equal because all box bodies appear to overlap well with each other with almost 50% pairwise overlap in all bodies. However, the errors vs. treatment groups plot suggests that the variance in BECK scores between the recent drug usage and long treatment length and recent drug usage short treatment length subjects is quite different, so we need to conduct the Brown-Forsythe test in order to make a strong conclusion about the treatment group variances.

We conduct both a Shapiro-Wilks test and Brown-Forsythe test in order to more rigorously assess the normality of errors and equal treatment group variances assumptions, respectively. Before conducting these two tests, we choose to use an alpha of 0.05 so as to avoid any post-hoc hypothesis test interpretations.

The null hypothesis of the SW test is that all of the errors are distributed normal with a mean of zero and standard deviation of (sigma)^2_epsilon, while the alternate hypothesis is that the errors are not distributed normal. At an alpha of 0.05, the Shapiro-Wilks test p-value of 0.0273 is less than alpha and so we reject the null hypothesis and conclude that the errors are not normally distributed. This result suggests that the normality of errors assumption has been violated.

The null hypothesis of the BF test is that all population treatment group variances are equal to each other (sigma_11)^2 = (sigma_12)^2=…, while the alternate hypothesis is that at least one population treatment group variance is not equal to that of the other treatment groups. At an alpha of 0.05, the Brown-Forsyte test p-value of 0.4267 is greater than alpha and so we fail to reject the null hypothesis and conclude that the equal treatment group variances assumption has not been violated.

We do not conduct a transformation of variables in order to satisfy the normality of errors assumption here.

We determine the best model as follows, with Factor A denoting the subjects’ history of recreational drug use and Factor B denoting the length of time over which the subjects have been receiving psychological treatment:

We first determine whether or not to include the interaction effect term in our model with; \[H_0: (\gamma\delta)_{ij}=0\ for\ all\ i,j \] \[H_A: at\ least\ one\ (\gamma\delta)_{ij}\neq0\ for\ an\ i,j \] At an alpha of 0.05, the p-value (0.8748) is greater than alpha so we fail to reject the null hypothesis and conclude that all interaction effect terms are equal to zero for all factor levels of the subjects’ history of drug use and the length of time that they have been receiving treatment. We thus conclude that the model that includes information on the the subjects’ history of drug use and length of time over which they have been receiving treatment is a statistically better fit than the model that includes information on both said factor effects and the interaction effect between the subjects’ history of drug use and treatment duration.

We then determine whether or not to include the effect of the subjects’ history of drug use to a model with the effect of the subject’s treatment length: \[H_0: \gamma_{i}=0\ for\ all\ i \] \[H_A: at\ least\ one\ \gamma_{i}\neq0\ for\ an\ i \]

At an alpha of 0.05, the p-value(0.0172) is less than alpha so we reject the null hypothesis and conclude that at least one effect of the subjects’ history of drug use is not equal to that of the others. Therefore, the model that includes information on both the subjects’ history of drug use and the length of time over which the subject has been receiving treatment is a statistically better fit than the model that only includes information on the length of time over which the subject has been receiving treatment.

We conclude by determining whether or not to include the effect of the length of time over which the subject has been receiving psychological treatment to a model with the effect of the subjects’ drug use history: \[H_0: \delta_{j}=0\ for\ all\ j \] \[H_A: at\ least\ one\ \delta_{j}\neq0\ for\ a\ j \]

At an alpha of 0.05, the p-value(0.0040) is less than alpha so we reject the null hypothesis and conclude that at least one effect of the subject’s treatment length is not equal to that of the others. Therefore, the model that includes information both on the subject’s treatment length and drug use history is a statistically better fit than the model that only includes information on the subject’s drug use history.

We conclude that the best model to use is the model that does not contain the interaction effects term and only contains both factor effects terms: \[Y_{ijk}=\mu_{..}+\gamma_{i}+\delta_{j}+\varepsilon_{ijk}\]

We calculated the following three conditional R^2 values:
1) When we add information about the interaction between the subjects’ recreational drug use history and length of psychological treatment to a model with information on both the subjects’ recreational drug use history and length of psychological treatment, there is an overall 0.14% reduction in error.
2) When we add information about the subjects’ recreational drug use history to a model with information on the subjects’ length of psychological treatment, there is an overall 2.86% reduction in error.
3) When we add information about the subjects’ length of psychological treatment to a model with information on the subjects’ recreational drug use history, there is an overall 5.48% reduction in error.

All confidence intervals use a confidence level of 95% and are corrected for with the Bonferroni multiplier with the number of confidence intervals taken (g) equal to 6.

Conclusion

We find that the model with the best statistical fit corresponds to the no-interactions, both factor effects model. We thus suggest that the factor effects of both the subjects’ treatment length and the subjects’ history of drug use be included. In the end, the fact that the confidence intervals that we calculated suggest no significant differences between treatment groups (both pairwise and contrast of three groups) supports the initial observations from the boxplots by treatment group; no treatment group appears to greatly differ from the others in both the centering and the spread of the BECK scores.

Partial.R2 = function(small.model,big.model){
  SSE1 = sum(small.model$residuals^2)
  SSE2 = sum(big.model$residuals^2)
  PR2 = (SSE1 - SSE2)/SSE1
  return(PR2)
}

find.mult = function(alpha,a,b,dfSSE,g,group){
  if(group == "A"){
    Tuk = round(qtukey(1-alpha,a,dfSSE)/sqrt(2),3)
    Bon = round(qt(1-alpha/(2*g), dfSSE ) ,3)
    Sch = round(sqrt((a-1)*qf(1-alpha, a-1, dfSSE)),3) 
  }else if(group == "B"){
    Tuk = round(qtukey(1-alpha,b,dfSSE)/sqrt(2),3)
    Bon = round(qt(1-alpha/(2*g), dfSSE ) ,3)
    Sch = round(sqrt((b-1)*qf(1-alpha, b-1, dfSSE)),3) 
  }else if(group == "AB"){
    Tuk = round(qtukey(1-alpha,a*b,dfSSE)/sqrt(2),3)
    Bon = round(qt(1-alpha/(2*g), dfSSE ) ,3)
    Sch = round(sqrt((a*b-1)*qf(1-alpha, a*b-1, dfSSE)),3) 
  }
  results = c(Bon, Tuk,Sch)
  names(results) = c("Bonferroni","Tukey","Scheffe")
  return(results)
}

scary.CI = function(the.data,MSE,equal.weights = TRUE,multiplier,group,cs){
   if(sum(cs) != 0 & sum(cs !=0 ) != 1){
    return("Error - you did not input a valid contrast")
  }else{
    the.means = find.means(the.data)
    the.ns =find.means(the.data,length)
    nt = nrow(the.data)
    a = length(unique(the.data[,2]))
    b = length(unique(the.data[,3]))
    if(group =="A"){
      if(equal.weights == TRUE){
        a.means = rowMeans(the.means$AB)
        est = sum(a.means*cs)
        mul = rowSums(1/the.ns$AB)
        SE = sqrt(MSE/b^2 * (sum(cs^2*mul)))
        N = names(a.means)[cs!=0]
        CS = paste("(",cs[cs!=0],")",sep = "")
        fancy = paste(paste(CS,N,sep =""),collapse = "+")
        names(est) = fancy
      } else{
        a.means = the.means$A
        est = sum(a.means*cs)
        SE = sqrt(MSE*sum(cs^2*(1/the.ns$A)))
        N = names(a.means)[cs!=0]
        CS = paste("(",cs[cs!=0],")",sep = "")
        fancy = paste(paste(CS,N,sep =""),collapse = "+")
        names(est) = fancy
      }
    }else if(group == "B"){
      if(equal.weights == TRUE){
        b.means = colMeans(the.means$AB)
        est = sum(b.means*cs)
        mul = colSums(1/the.ns$AB)
        SE = sqrt(MSE/a^2 * (sum(cs^2*mul)))
        N = names(b.means)[cs!=0]
        CS = paste("(",cs[cs!=0],")",sep = "")
        fancy = paste(paste(CS,N,sep =""),collapse = "+")
        names(est) = fancy
      } else{
        b.means = the.means$B
        est = sum(b.means*cs)
        SE = sqrt(MSE*sum(cs^2*(1/the.ns$B)))
        N = names(b.means)[cs!=0]
        CS = paste("(",cs[cs!=0],")",sep = "")
        fancy = paste(paste(CS,N,sep =""),collapse = "+")
        names(est) = fancy
      }
    } else if(group == "AB"){
      est = sum(cs*the.means$AB)
      SE = sqrt(MSE*sum(cs^2/the.ns$AB))
      names(est) = "someAB"
    }
    the.CI = est + c(-1,1)*multiplier*SE
    results = c(est,the.CI)
    names(results) = c(names(est),"lower bound","upper bound")
    return(results)
  }
}
